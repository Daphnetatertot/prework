Full Stack Starter Notes

CSS

  • CSS stands for cascading style sheet
  • You can use CSS for basic styling on you page like color, font, font style etc.
  • CSS operates using a box format called the box model
  • The box model is based off of the use of 4 properties

Git

  • Git is a place to store code
  • Each project needs its own repository
  • A branch or repository can be created right from a terminal on newer macOS
  • Git has Git hub and Git Lab, but the main one needed for a beginner is

JavaScript

  • JavaScript is used essentially to trigger events
  • Events are things like what a button is supposed to do when it's clicked
  • To use JavaScript in HTML you have to link it using a script tag at the bottom of the body element
  • JavaScript has multiple API's to help reduce the amount of code needed
  • JavaScript is being constantly update

HTML

  • HTML is short for HyperText Markup Language
  • HTML has elements that use tags to describe them
  • A doc type must be declared at the top of the HTML page to tell the browser what it's working with
  • There are 6 headings and h1 can only be used once
  • HTML is the building block for the majority of all webpages, and creates the basic structure for web applications
  • You can link outside images, URLs, Stylesheets, etc. right into HTML. The multiple documents keeps each line clean and formatted and easy to read
